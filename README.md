google-calendar-widget
===
##概要
Wordpressでグーグルカレンダーを取り込むプラグイン

##使い方
1. 管理画面の"インストール済みプラグイン"から"Google Calendar Widget"を有効化  
2. 管理画面の"外観"->"ウィジェット"で"Google Calendar"を"ウィジェットエリア"にドラッグアンドドロップ  
3. 読み込むカレンダーのID指定などプラグインの設定をする
4. サイトを表示
